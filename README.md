# Care Hospital Landing Page

A modern and responsive hospital landing page built to showcase healthcare services, doctors, and patient-friendly features.

## 🚀 Features

* Responsive Design
* Modern UI/UX
* Hero Section
* Doctor Cards
* Services Section
* Testimonials
* Mobile Friendly Layout
* Clean Footer Section

## 🛠️ Technologies Used

* Tailwind CSS
* Next.js

## 📱 Responsive Design

This project is fully responsive and optimized for:

* Desktop
* Tablet
* Mobile Devices



## 🎯 Purpose of the Project

This project was created to practice:

* Frontend Development
* Responsive Design
* UI Design
* Component-Based Architecture
* Modern Web Development Workflow
